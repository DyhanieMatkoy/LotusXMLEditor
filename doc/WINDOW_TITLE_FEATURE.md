# Window Title Feature

## Описание
Заголовок окна приложения теперь показывает имя редактируемого файла перед основным названием приложения.

## Формат заголовка

### Когда файл открыт:
```
имя_файла.xml - Lotus Xml Editor - Python Version
```

### Когда файл не открыт:
```
Lotus Xml Editor - Python Version
```

## Примеры

### Обычный файл:
```
document.xml - Lotus Xml Editor - Python Version
```

### Файл с кириллическим именем:
```
документ.xml - Lotus Xml Editor - Python Version
```

### Файл с длинным путем:
```
my_long_filename.xml - Lotus Xml Editor - Python Version
```

## Когда обновляется заголовок

Заголовок окна автоматически обновляется в следующих случаях:

1. **При открытии файла** - показывает имя открытого файла
2. **При создании нового файла** - показывает базовый заголовок
3. **При сохранении файла как...** - показывает новое имя файла
4. **При реконструкции XML** - показывает базовый заголовок

## Техническая реализация

### Метод `_update_window_title()`
Централизованный метод для обновления заголовка окна:

```python
def _update_window_title(self):
    """Update window title with current file name"""
    base_title = "Lotus Xml Editor - Python Version"
    if self.current_file:
        filename = os.path.basename(self.current_file)
        self.setWindowTitle(f"{filename} - {base_title}")
    else:
        self.setWindowTitle(base_title)
```

### Использование
Вместо прямого вызова `setWindowTitle()` теперь используется `_update_window_title()` во всех местах где нужно обновить заголовок.

## Преимущества

1. **Удобство** - сразу видно какой файл редактируется
2. **Многооконность** - при открытии нескольких окон легко различить файлы
3. **Консистентность** - единый формат заголовка во всем приложении
4. **Поддержка Unicode** - корректно отображает файлы с кириллическими именами

## Совместимость

Функция работает со всеми типами файлов и путей:
- Короткие имена файлов
- Длинные пути
- Кириллические имена
- Специальные символы в именах файлов
