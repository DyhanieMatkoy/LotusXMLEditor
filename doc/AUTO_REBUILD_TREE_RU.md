# Опция AutoRebuildTree

## Что это?

Новая опция **AutoRebuildTree** позволяет контролировать, будет ли дерево XML автоматически обновляться при изменении текста в редакторе.

## Как это работает?

### Опция включена (по умолчанию) ✓
- Дерево обновляется автоматически через 5 секунд после изменения текста
- Удобно для быстрого редактирования небольших файлов

### Опция выключена ⚠
- Дерево НЕ обновляется автоматически
- Слева от кнопки **Rebuild Tree** появляется оранжевый индикатор **⚠**
- Индикатор показывает, что дерево нужно обновить вручную
- Нажмите **Rebuild Tree** (или **F11**) для обновления дерева
- После обновления индикатор исчезает

## Как настроить?

1. Откройте **Settings** → **Settings...**
2. Найдите раздел **Tree Updates**
3. Установите или снимите флажок **Auto Rebuild Tree**
4. Нажмите **OK**

## Когда использовать?

### Выключите опцию, если:
- Работаете с большими XML файлами (медленное обновление)
- Делаете много изменений подряд
- Хотите контролировать момент обновления дерева

### Оставьте включенной, если:
- Работаете с небольшими файлами
- Хотите видеть изменения в дереве сразу
- Не испытываете проблем с производительностью

## Горячие клавиши

- **F11** - Обновить дерево вручную (Rebuild Tree)

## Визуальный индикатор

Когда дерево требует обновления, вы увидите:

```
[⚠] [Rebuild Tree] [Enable Sync] ...
```

Индикатор **⚠** исчезнет после нажатия на **Rebuild Tree**.
