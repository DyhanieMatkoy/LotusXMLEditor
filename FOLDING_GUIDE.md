# Code Folding Guide

## Визуальные контролы сворачивания

В редакторе теперь доступны визуальные индикаторы сворачивания кода в области номеров строк:

### Индикаторы:
- **Треугольник вправо (►)** - свернутый блок кода
- **Треугольник вниз (▼)** - блок, который можно свернуть (появляется при наведении мыши)

### Использование:
1. Наведите мышь на номер строки с XML-элементом
2. Появится треугольник вниз - кликните, чтобы свернуть
3. Свернутый блок отображается с треугольником вправо - кликните, чтобы развернуть

## Горячие клавиши

| Клавиши | Действие |
|---------|----------|
| `Ctrl+Shift+[` | Свернуть текущий XML-элемент под курсором |
| `Ctrl+Shift+]` | Развернуть текущий XML-элемент под курсором |
| `Ctrl+Shift+U` | Развернуть все свернутые элементы |

## Меню

Все команды сворачивания доступны в меню **View**:
- Fold Current Element
- Unfold Current Element
- Unfold All

## Примечания

- Сворачивание работает только для корректно сформированных XML-элементов
- При редактировании текста все свернутые блоки автоматически разворачиваются
- Номера строк должны быть включены (`Ctrl+L`) для отображения контролов сворачивания
